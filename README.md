This is a weathergotchi or a pocket weather deviec, its a device that stays always on and shows you the weather, its super simple, it will have a oled screen on the front that uses the wifi on the esp32 to conect to the interent, call an weather api and show you the waether of your location.


To set it up, i will have it show a qr code on its screen that links to an external web page where you can put in the login info for the wifi.

To assemle it, we must screw in the screen with the included screws and then solder the wires from the esp32 to the screen and also connect the battery with the jst connector
